ターゲットタイムでピタッと止める、シンプルかつ中毒性のあるストップウォッチゲーム。

## 🚀 ライブデモ
[👉 ここをクリックしてゲームを遊ぶ](https://bbbb199602111220109.github.io/just_stop_game/)

## ✨ 特徴
- **洗練されたUI**: サイバーパンクな雰囲気を感じさせるダークモード・ネオンデザイン。
- **直感的な操作**: 1つのボタンでスタート・ストップ・リトライまで完結。
- **モバイル最適化**: スマホのタップ操作でも遅延なく快適に遊べる設計。
- **ランダムな目標設定**: 毎回変わるターゲットタイム（3.0〜8.0秒）で飽きさせない設計。

## 🛠️ 使用技術
- **HTML5 / CSS3**: モダンなフレックスボックスとアニメーションを使用。
- **JavaScript**: ブラウザの `requestAnimationFrame` を活用し、滑らかで正確なタイマー処理を実現。

## 💡 こだわりポイント
- **UXの追求**: ボタンを押した瞬間のリアクション（スケール変化）や、成功時の演出にこだわり、短いプレイ時間でも満足感を得られるようにしました。
- **レスポンシブ**: PCでもスマホでも、常に中央にバランスよく表示されるレイアウト。
